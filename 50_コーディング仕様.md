# メディアサイト コーディング仕様

## 基本設定

| 項目 | 値 |
|------|-----|
| サイトタイプ | スニーカー専門ニュースメディア |
| 設定単価 | コーディング（トップページのみ）：12万円 |
| レイアウト | 1カラム（全幅） |
| レスポンシブ | 対応必須 |

---

## 各セクション仕様

### Header
- ロゴ（h1タグ）とハンバーガーメニューを配置
- Flexboxで両端揃え（`justify-content: space-between`）
- **ハンバーガーメニュー**
  - PC時、スマホ時ともにハンバーガーメニューを使用
  - navタグ内にメニューを記述
  - 3本線を表現するクラスを作成
  - メニューオープン時の全体マスク用クラスを作成
  - jQueryでクリック時のオープン/クローズを切り替え
- **ナビゲーションメニュー**
  - PICK UP、FEATURE、CONTACT の3項目
  - クリック時はスムーススクロールで各セクションに移動
- **SNSリンク**
  - Twitter、Facebook、Instagramへのリンク

### 動画エリア（#video）
- 全画面表示（width: 100%, height: ブラウザの高さに可変）
- videoタグを使用して動画を埋め込み
- videoタグの属性設定:
  - `autoplay`: 自動再生
  - `loop`: 繰り返し再生
  - `muted`: 消音
  - `playsinline`: インライン再生
- 動画ファイル: `video/video.mp4`

### PICK UP（#pickup）
- sectionタグで囲む
- セクションタイトル「PICK UP」
- **カルーセルスライダー**
  - jQueryプラグイン「slick」を使用
  - 9枚の画像（pickup1.jpg ～ pickup9.jpg）
  - 自動スライド表示
  - 画像には代替テキスト（alt属性）を設定

### FEATURE（#feature）
- sectionタグで囲む
- セクションタイトル「FEATURE」
- **グリッドレイアウト**
  - CSS Grid（`display: grid`）を使用
  - `grid-template-columns` で画像の並べ方とサイズを指定
  - タイル状に画像を配置（9枚の画像）
- **各記事リスト**
  - 画像（feature1.jpg ～ feature9.jpg）
  - カテゴリ表示（"category"）
  - 記事テキスト
  - 日付（XXXX.XX.XX形式）
- **画像フェード表示**
  - スクロールイベントで画像がビューポートに入ったタイミングで表示
  - opacityとtransformを使ってフェード表示
  - ふわっと表示されるアニメーション効果

### CONTACT（#contact）
- sectionタグで囲む
- セクションタイトル「CONTACT」
- 説明文（テキストテキストテキスト...）
- **コンタクトフォーム**
  - formタグで囲む
  - dlタグで構成:
    - dtタグ: ラベル（Name、Mail、Message）
    - ddタグ: 入力フィールド
  - 入力項目:
    - Name: テキスト入力（inputタグ、type="text"）
    - Mail: メール入力（inputタグ、type="email"）
    - Message: テキストエリア（textareaタグ）
  - 送信ボタン（inputタグ、type="submit"）
  - ボタンホバー時に色反転（`:hover`で実装）
  - ※送信機能は不要

### Footer
- footerタグで囲む
- コピーライト表示: `© Sneakers`
- pタグで記述

## 技術要件

### HTML
- セマンティックHTML（header、nav、section、footer等）を適切に使用
- videoタグの属性を正しく設定
- formタグでフォームを構成
- h1、h2タグで見出しを適切に設定

### CSS
- Flexbox: ヘッダーのレイアウト
- CSS Grid: FEATUREセクションのグリッドレイアウト
- Position: ハンバーガーメニューの配置
- Transform, Opacity: 画像フェードアニメーション
- Hover効果: ボタンの色反転

### JavaScript（jQuery）
- ハンバーガーメニューの開閉
- スムーススクロール
- カルーセルスライダー（slickプラグイン）
- スクロールイベント（画像フェード表示）

## レスポンシブ対応

デモサイトの動作を参考に、PC/SP両対応でレイアウトを調整

```css
/* PC */
@media (min-width: 768px) {
  /* PC用スタイル */
}

/* SP */
@media (max-width: 767px) {
  /* SP用スタイル */
  /* グリッドレイアウトのカラム数を調整 */
}
```

## 実装の流れ（推奨）

1. **HTML構造の作成**
   - セマンティックHTMLでセクション分け
   - videoタグの設置
   - フォームの実装

2. **CSS基本スタイル**
   - リセットCSS適用
   - 全体のレイアウト設定
   - 各セクションのスタイリング

3. **グリッドレイアウト**
   - CSS Gridの実装
   - レスポンシブ対応

4. **jQuery実装**
   - ハンバーガーメニュー
   - スムーススクロール
   - slickプラグインの設定
   - 画像フェードアニメーション

5. **動作確認**
   - 各機能のテスト
   - レスポンシブ確認
   - ブラウザ互換性チェック
